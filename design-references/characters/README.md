# 人物設計參考截圖

僅供本機對照版型，**不是**可直接用的 sprite 素材。

| 檔案 | 來源 |
|------|------|
| `og/game2_*.gif` | [beanfun 角色介紹](https://tw.beanfun.com/bnb/game2.htm) 官方 GIF |
| `og/*_og_ref.png` | 上述 GIF 裁切後的 96px 參考 |
| `official_beanfun_characters.png` | beanfun 角色頁全頁截圖 |
| `official_mlwd_personage.png` | [神奇世界 角色頁](http://mlwd.com.tw/BNB/personage.htm) |
| `official_beanfun_home.png` | [beanfun 爆爆王首頁](https://tw.beanfun.com/bnb/) |
| `ref_map_overview.png` | 專案內 `reference-village10.png` 全圖 |
| `ref_char_*.png` | 村10 截圖內單角色裁切 |

下載 / 更新 OG 參考：

```bash
python3 scripts/download-og-character-refs.py
node scripts/capture-character-refs.mjs
```
